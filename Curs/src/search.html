Поиск

В главном окне поле "Поиск" предназначено для поиска нужного рецепта. Достаточно ввести текст, поиск происходит автоматически при редактировании текста в поле поиска. Доступен поиск по фильтрам. Для выбора фильтра нужно нажать на кнопку "Фильтры". Будет открыто меню с доступными фильтрами. Фильтр определяет где имеено будет происходить поиск по тексту из поля поиска. При выборе фильтра "Название", поиск текста будет происходить в названиях рецептов. Аналогично работают остальные фильтры. При выборе фильтра "Глобальный поиск", будет осуществляться поиск заданного текста в названиях, ингредиентах и с тексте рецепта.

* По умолчанию стоит фильтр "Глобальный поиск". Все найденные результаты будут отображены в поле с названиями рецептов. Если поле пустое, рецептов, соответсвующим критериям поиска, найдено не было. Если в результате поиска были найдены рецепты, можно выбрать отображение по категории и отображение избранных рецептов.
