Создание рецептов

Кнопка "Новый рецепт" открывает окно создания нового рецепта. В поле "Название" нужно вписать название вашего рецепта. В поле "Ингредиенты" вписать необходимые для рецепта ингредиенты. В поле "Рецепт" необходимо вписать рецепт. Кнопка "Загрузить изображение" открывет окно выбора изображения, которое будет сопровождать созданный рецепт. Кнопка "Категория" вызывает меню для выбора категории рецепта. Кнопка "Сохранить рецепт" сохраняет рецепт. Кнопка "Отменить" закрывает окно создания рецепта, с сохранением уже заполненных полей. Если рецепт не сохранить, то при закрытии приложения данные о рецепте будут утеряны.

* Если заполнить не все поля, не добавление рецепта будет невозможно и приложение выдаст уведомление о том, что не все поля заполнены. Поле с изображением единственное необязательное. При отсутствии изображения рецепта, рецепт будет сопровождать изображение по умолчанию.
